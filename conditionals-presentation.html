First Conditional
2 / 7
Real & Possible
The First Conditional
Use for situations that are real and genuinely possible in the present or future.
IF + subject + present simple  ,  subject + will / can / may / might + infinitive
If-clause first → comma
If I have time, I will go to the market.
If you book early, you will save money.
Result first → no comma
She will get cold if she doesn't wear a coat.
You will save money if you book early.
Other modals in the result
If we hurry, we can catch the last train. (ability)
If it is sunny, we might go to Brighton. (possibility)
🇬🇧 UK Guide context
"If you visit Liverpool, you will see the Albert Dock."
"If we hurry, we can catch the last Tube."
⚠️ Key Rule
❌ "If you will visit London, you will love it."
✅ "If you visit London, you will love it." — Never use will in the if-clause!

Second Conditional

Hypothetical & Imaginary
The Second Conditional
Use for situations that are imaginary, unlikely or impossible right now.
IF + subject + past simple  ,  subject + would / could / might + infinitive
If-clause first → comma
If I won the lottery, I would buy a big house.
If I lived in London, I would cycle everywhere.
Result first → no comma
Grandma would be lonely if I didn't visit her.
She would eat cake if you made one.
If I WERE (not was)
Use were instead of "was" for all persons — especially in formal English.
If I were you, I would take the train.
If she were PM, museums would be free.
Forming questions
(Wh-word) + would + subject + inf. + if + subject + past simple
What would you do if you lost your passport?
⚠️ Key Rule
❌ "If I would have more money, I would travel more."
✅ "If I had more money, I would travel more." — Never use would in the if-clause!

First vs Second

Comparison
First vs Second — The Key Difference
The key is how real or likely the situation is — not just the verb tense.
🔵 First Conditional 🟣 Second Conditional
Meaning Real and possible — it could happen Hypothetical, unlikely or imaginary
If-clause IF + Present Simple IF + Past Simple
Result will / can / may would / could / might
Example "If you visit Edinburgh, you will love it." (realistic) "If I were Queen, I would open all palaces for free." (imaginary)
Comma Both: comma when if-clause is first. No comma when result is first.
🔵 First Conditional errors
❌ "If it will rain, we will cancel." — no will in if-clause!
✅ "If it rains, we will cancel."
🟣 Second Conditional errors
❌ "If I would have more time, I'd travel more."
✅ "If I had more time, I'd travel more."

Unless & Variations

Extra Structures
Unless & Variations
Special connectors that add nuance — they work with both First and Second Conditional.
UNLESS = IF NOT
Unless
Means "if not / except if". Verb after unless stays positive — the negative meaning is already included.

We'll miss the bus unless we hurry.
(= if we don't hurry)
Unless you book now, you won't get tickets.
CONDITION REQUIRED
As long as / Provided that
Means "on the condition that" — the condition must be met.

You can go as long as you're back by midnight.

Provided that you have a valid passport, you can enter the UK.
PRECAUTION
In case
Means "because it might happen" — you act before the event as a precaution.

Take an umbrella in case it rains.

Save the number in case you get lost.
⚠️ Different from "if" — in case = before the event
🇬🇧 UK Guide — all three in context
unless: "Unless you book in advance, you won't get Wimbledon tickets."
in case: "Save the guide's number in case you get separated."
as long as: "You can join the tour as long as you arrive before 9 AM."

Common Errors

Watch Out!
Common Errors
These are the most frequent mistakes. Read each pair and understand why the wrong version is wrong.
Error 1
WILL in the if-clause (1st Conditional)
❌ "If you will visit Edinburgh, you will love it."
✅ "If you visit Edinburgh, you will love it."
The if-clause always uses present simple. Never use will in the if-clause.
Error 2
WOULD in the if-clause (2nd Conditional)
❌ "If I would have more money, I would travel more."
✅ "If I had more money, I would travel more."
The if-clause uses past simple. Only the result clause uses would.
Error 3
Using 1st when you mean 2nd (reality check)
❌ "If I am a millionaire, I will buy a castle." (you are NOT one)
✅ "If I were a millionaire, I would buy a castle."
If the situation is imaginary or impossible, use the 2nd Conditional.
Error 4
Comma confusion
❌ "You will get wet, if you don't take an umbrella."
✅ "If you don't take an umbrella, you will get wet." (comma: if-clause first)
✅ "You will get wet if you don't take an umbrella." (no comma: result first)
